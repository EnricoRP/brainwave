## 🔹Instalasi Vite
- Create Project
    `npm create vite@latest ./ -- --template react`
    `npm i`
- Install Tailwind
    `npm install -D tailwindcss@3 postcss autoprefixer`
    `npx tailwindcss init -p` : membuat file config (tailwind.config.js & postcss.config.js)
- Configure template paths
    ```typescript
        content: [
        "./index.html",
        "./src/**/*.{js,ts,jsx,tsx}",
        ]
    ```
